.. _Usage:

Usage
=====

This is the usage guide for **Neuro-Fuzzy Toolbox** for building and training
neuro-fuzzy models.

If you have not yet installed the package, follow the instructions in
:ref:`Installation`.

.. important::

   **Neuro-Fuzzy Toolbox** is built on top of **PyTorch**, so a basic familiarity
   with this library is recommended for a better understanding of its usage
   (see `PyTorch basics <https://docs.pytorch.org/tutorials/beginner/basics/intro.html>`_).
