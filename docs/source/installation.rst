.. _Installation:

Installation
============

Para la instalación de Neuro-Fuzzy Toolbox, se recomienda el uso de un entorno virtual de Python. A continuación se describen los pasos para instalar la herramienta creando un entorno
virtual y utilizando el gestor de paquetes de Python. Para más información sobre la creación de entornos virtuales, consulta la `documentación oficial <https://docs.python.org/3/library/venv.html>`_.

Puedes instalar Neuro-Fuzzy Toolbox de la siguiente manera:

- clona el repositorio de GitHub y dirígete a la carpeta del proyecto,

.. code-block:: bash

   git clone https://github.com/edge-mirage/neuro-fuzzy-toolbox.git
   cd neuro_fuzzy_toolbox

- crea un entorno virtual y actívalo,

.. code-block:: bash

   python3 -m venv env_name
   source env_name/bin/activate

- e instala las dependencias del proyecto en el entorno virtual.

.. code-block:: bash

   pip install -r requirements.txt

Para más información sobre el uso de la herramienta, consulta la sección :ref:`Usage`.
