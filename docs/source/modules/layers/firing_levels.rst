Firing Levels Layers
====================

En este módulo se encuentran las capas asociadas a los niveles de disparo implementadas en el Toolbox (incluyendo la capa de normalización).
Estas capas se utilizan para calcular los niveles de disparo de los conjuntos difusos en base a los valores de pertenencia emulando operaciones AND entre las variables lingüisticas correspondientes a los features de los datos de entrada.
