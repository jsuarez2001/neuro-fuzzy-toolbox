.. _Membership Functions:

Membership Functions
====================

This module defines the membership functions (MFs) used in the toolbox.
MFs are used to define the fuzzy sets of the input variables in
neuro-fuzzy models.

All classes below inherit from the base class ``MembershipFunction``,
which defines the common interface and serves as a reference for future
implementations.
